// DỮ LIỆU TRỐNG

Chọn bộ lọc hoặc đợi hệ thống tải dữ liệu.

Server Hop hoạt động như thế nào?

Khi bạn nhấn nút VÀO PHÒNG, hệ thống sẽ tự động kích hoạt ứng dụng Roblox trên máy tính của bạn và trực tiếp kết nối bạn vào phòng chơi tương ứng.

Lựa chọn "Sao chép mã Console" dùng để làm gì?

Nếu trình duyệt chặn link trực tiếp, hãy nhấn nút copy trên thẻ phòng. Mở Console (F12) ở trang Roblox game Catch a Monster, dán đoạn mã vừa copy và nhấn Enter để vào game ngay lập tức.

1. KHỞI CHẠY HỆ THỐNG CÀY CUỐC

Đảm bảo **Roblox Client** trên máy tính của bạn đã được cài đặt sẵn. Bật tool Sv_Hops trên máy chủ và truy cập trang điều khiển qua trình duyệt để bắt đầu cày Boss.

2. CƠ CHẾ SĂN BOSS RIFT (MỖI 15 PHÚT)

Trong tựa game Catch a Monster, Boss Rift sẽ xuất hiện tuần hoàn theo mốc thời gian UTC thực tế:

  • 🔥 Xuất hiện chính xác tại các phút: :00, :15, :30, :45.
  • 🌟 Đặc biệt: Đầu mỗi giờ (ví dụ 10:00, 11:00) sẽ spawn Boss Giờ Cực Hiếm có giá trị phần thưởng siêu cao.
  • ⏱️ Theo dõi bảng **ĐẾM NGƯỢC BOSS RIFT** phía trên để nhảy server trước 1-2 phút.

3. BÍ QUYẾT SỬ DỤNG AUTO SERVER HOP

Tích chọn **Auto Hop** bên thanh Sidebar để hệ thống tự động nhảy sang server vắng người kế tiếp sau mỗi khoảng thời gian cài đặt. Rất phù hợp khi bạn vào phòng nhưng không thấy Boss và muốn chuyển phòng ngay lập tức để tìm Boss ở máy chủ khác!

4. PHƯƠNG PHÁP CHẠY LỆNH CONSOLE (TRÁNH LỖI TRÌNH DUYỆT)

Nếu trình duyệt của bạn chặn không cho mở ứng dụng Roblox tự động, hãy làm theo hướng dẫn 3 bước sau:

  1. Nhấn vào biểu tượng **Copy màu xanh** trên thẻ Server bạn muốn vào để lưu mã console vào clipboard.
  2. Truy cập trang game **Catch a Monster** trên web Roblox, nhấn phím F12 (hoặc nhấn chuột phải chọn Kiểm tra / Inspect -> chuyển sang tab Console).
  3. Dán đoạn mã (Ctrl + V) vào khung Console và nhấn Enter. Trình duyệt sẽ đưa bạn thẳng vào game trong nháy mắt!
